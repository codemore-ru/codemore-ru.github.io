Для разработки программного обеспечения используются среды разработки, которые включают в себя текстовый редактор, компилятор, компоновщик и отладчик.

Для выполнения практических работ в рамках учебного курса будет описана работа с программой Keil mVision версии 5.14, более новую версию можно загрузить с официального сайта (www.keil.com/download/product/). Программа позволяет разрабатывать программное обеспечение под множество микроконтроллеров, в том числе и под микроконтроллеры компании «Миландр» (milandr.ru).

Для целей обучения может использоваться пробная версия Keil, которая позволяет компилировать программы, которые образуют код, объёмом до 32 кибибайт.

Для использования кириллицы в Keil следует воспользоваться подменой файла UvEdit.dll, разработанной ______________.

Установка среды разработки

Запустите файл mdk514.exe (прилагается на диске).

В появившемся окне программы Setup MDK ARM V5.14 нажмите кнопку Next.

Далее поставьте галочку напротив надписи I agree to all terms of the preceding License Agreement (соглашение с условиями лицензии). Нажмите кнопку Next.

Далее программа установки предложит выбрать путь для установки программы C:\Keil_v5 и путь к пакетам C:\Keil_v5\ARM\PACK. Предлагается оставить всё как есть, и нажать кнопку Next.

Затем программа попросит ввести First Name (имя), Last Name (фамилия), Company Name (название компании) и E-mail (адрес электронной почты). Введите данные и нажмите Next.

Далее программа установки начнёт процесс инсталляции файлов на компьютер, а затем распаковку пакетов и драйверов.

В конце установки снимите флажок напротив надписи Show Release Notes и нажмите кнопку Finish, установка программы завершена.

Установка пакетов

Для возможности загрузки разрабатываемых программ в микроконтроллер компании «Миландр» необходимо запустить файл mdr_spl_v1.4.1.pack.

Для возможности в среде разработки печатать буквы кириллицы необходимо файл UvEdit.dll, находящийся в папке C:\Keil_v5\UV4, заменить на одноимённый ему, UvEdit.dll.

Настройка среды разработки

Запустите среду разработки uVision. Для удобства работы разверните окно. Пройдите по пути Edit->Configuration и в появившемся окне перейдите на вкладку Colors & Fonts.

В окне Window выберите C/C++ Editor files и затем выберите шрифт Courier вместо Courier New (это необходимо для возможности печатать и отображать буквы кириллицы). Нажмите OK.
