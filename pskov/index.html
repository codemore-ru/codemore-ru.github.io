Олимпиада по спортивному программированию ПсковГУ - 2021

Дата проведения: 10.04.2021

Время проведения: 11:00 - 17:00

Место проведения: ул. Льва Толстого, д. 4, корп. 2, ауд. 24

Регистрация

Для регистрации необходимо сообщить фамилию, имя, отчество, номер группы, предпочитаемый язык и среду программирования на кафедру ИКТ (ул. Льва Толстого, д. 4, корп. 2, ауд. 26), или на е-mail kafikt@pskgu.ru, или по телефону 79-76-99.

Регистрация нужна для подготовки нужного количества рабочих мест и подготовки списков участников.

Регистрация открыта до 09.04.2021 включительно

Языки программирования и компиляторы

В проверяющей системе доступны:

  • fpc - Free Pascal 2.6.4
  • gcc - GNU C 4.9.2
  • g++ - GNU C++ 4.9.2
  • python - Python 2.7.9
  • python3 - Python3 3.4.2
  • java - Java JDK 1.7.0_121
