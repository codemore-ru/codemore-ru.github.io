Олимпиада ПсковГУ по спортивному программированию

Цель мероприятия: повышение интереса учащихся к программированию и данному виду спорта.

Дата проведения: 17 декабря 2022 года.

Время проведения:

  • 10:00 - 10:30 - Информация о соревновании
  • 10:40 - 11:20 - Пробный тур
  • 11:30 - 15:30 - Основной тур
  • 15:50 - 17:00 - Разбор задач и награждение победителей

Регистрация

Для регистрации необходимо сообщить фамилию, имя, отчество, номер группы, номер действующего мобильного телефона и адрес электронной почты, предпочитаемый язык и среду программирования на кафедру ИКТ (ул. Льва Толстого, д. 4, корп. 2, ауд. 26), или на е-mail kafikt@pskgu.ru, или по телефону 79-76-99.

Регистрация нужна для проведения организационных и технических мероприятий.

Регистрация открыта до 16 декабря 2022 года включительно

Языки программирования и компиляторы

В проверяющей системе доступны:

  • fpc - Free Pascal 3.0.4
  • gcc - GNU C 9.3.0 (-std=gnu11)
  • g++ - GNU C++ 9.3.0 (-std=gnu++11)
  • python - Python 2.7.18
  • python3 - Python3 3.8.10
  • java - OpenJDK 11
